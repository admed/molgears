MolGears's
====================================

The Molgears is a database tool for storing chemical compounds, calculating descriptors, tracking synthesis, storing analytical and bioactivity data.
Bioactivity is reported in IC50 results. Data can be filtered, analyzed and downloaded to various file formats.

See :ref:`Overview` to learn more.

Molgears is accessed via a web interface. See :ref:`getting_started` to look at interface.
